Istio Gateway 统一入口

# 🎯 目标（统一入口设计）

最终效果：

          http://192.168.51.100（唯一）
                       |
                 Istio Gateway
                       |
        --------------------------------
        |              |               |
   grafana        vmagent        logs
   HTTPRoute      HTTPRoute      HTTPRoute

# 🧠 核心设计原则（生产标准）

# ✔ 1. 只保留一个 Gateway（入口层）

  • 只用：istio-ingressgateway
  • 或一个专用 gateway（推荐）

# ✔ 2. 所有业务只写 HTTPRoute

  • 不再创建多个 Gateway IP

# ✔ 3. 所有域名统一入口解析

*.zzppjj.top → 192.168.51.100

# ✔ 4. 按 host + path 分流

# 🚀 方案设计（推荐你用这个）

# 统一 TLS（核心）

# 🎯 目标

所有访问变成：

https://*.zzppjj.top
1

# 🔐 第 1 步：创建统一 TLS 证书（自签 or CA）

# ✔ 方式 A（推荐生产）：CA证书

如果你有企业 CA，直接签：

# ✔ 方式 B（测试/内网）：自签证书

openssl req -x509 -nodes -days 365 \
-newkey rsa:2048 \
-keyout tls.key \
-out tls.crt \
-subj "/CN=*.zzppjj.top"
1
2
3
4
5

# 创建 secret

kubectl create -n istio-system secret tls wildcard-zzppjj-tls \
--key=tls.key \
--cert=tls.crt

# 🧱 第 2 步：升级 Gateway（HTTP → HTTPS）

👉 替换你的 istio-unified-gateway

apiVersion: gateway.networking.k8s.io/v1
kind: Gateway
metadata:
  name: istio-unified-gateway
  namespace: istio-system
spec:
  gatewayClassName: istio
  listeners:

  # HTTP（自动跳 HTTPS）
  - name: http
    port: 80
    protocol: HTTP
    hostname: "*.zzppjj.top"
    allowedRoutes:
      namespaces:
        from: All

  # HTTPS（核心）
  - name: https
    port: 443
    protocol: HTTPS
    hostname: "*.zzppjj.top"
    tls:
      mode: Terminate
      certificateRefs:
      - name: wildcard-zzppjj-tls
        kind: Secret
    allowedRoutes:
      namespaces:
        from: All

# 🌐 第 3 步：HTTP 自动跳 HTTPS（推荐）

创建 HTTPRoute（全局跳转）

apiVersion: gateway.networking.k8s.io/v1
kind: HTTPRoute
metadata:
  name: redirect-to-https
  namespace: istio-system
spec:
  parentRefs:
  - name: istio-unified-gateway
  hostnames:
  - "*.zzppjj.top"
  rules:
  - matches:
    - path:
        type: PathPrefix
        value: /
    filters:
    - type: RequestRedirect
      requestRedirect:
        scheme: https
        statusCode: 301

# 📦 2. 灰度 HTTPRoute（权重分流）

apiVersion: gateway.networking.k8s.io/v1
kind: HTTPRoute
metadata:
  name: vmagent-gray-route
  namespace: monitoring
spec:
  parentRefs:
  - name: istio-unified-gateway
    namespace: istio-system
    sectionName: https

  hostnames:
  - vmagent.zzppjj.top

  rules:
  # 🎯 10% 灰度流量
  - matches:
    - headers:
      - name: x-canary
        value: "true"
    backendRefs:
    - name: vmagent-vmagent
      port: 8429
      weight: 10

  # 🎯 默认 90% stable
  - backendRefs:
    - name: vmagent-vmagent
      port: 8429
      weight: 90
