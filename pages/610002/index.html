工作经历回忆

# 1、高中时期

​ 高一下学期叛逆不想读书，跟父母去深圳打工，在工地搬砖，主要是搬运水泥和搅拌水泥浆，协助帮忙做苦力活。干过一个多月感觉吃不消放弃了就回到学校去继续读书

​ 总结：想出去工作想法很简单，不想在校园呆了，受够了学习、被人监督管理。想要自由，当时的想法很简单单纯、对社会上的事一无所知，以为自己能干一番天地。实际去了之后发现很迷茫，也没有得到想要的自由，还是被父母管束。最后妥协了就回去继续上学了

​

# 2、大学时期

​ 2.1、做过派发传单、在校门口摆摊卖苹果、国庆期间在酒店厨房做传菜员。

​ 2.2、暑假期间在深圳做过两个月保安。其它同学去工厂了我没有去。 ​

​ 总结：大学时光还是非常快乐的，自由自在。彻底放飞自我，基本是每天到泡网吧，要么是和同学出去游玩。度过一个快乐时光。虽然没有学习到什么知识，但我从来没后悔过

# 3、实习期

​ 3.1、在武汉做不到一个月的网优工程师，就是去一些基站和大型商超测试信号、去一些工厂把影响信号源的私人装的信号源设备偷偷装过滤波源的装置。

​ 3.2、去深圳中兴工厂干了三个月的流水线

​ 3.3、去浙江绍兴公司干了半年，主要业务是网上卖菜，在里面做销售、一些其它杂事。

​ 3.4、考了一个驾照

​ 3.5、去中山干了半年的宽带安装

总结：这次是真的要为生计考虑了，父母不在提供资金上帮助，为了生存我必须找工作干活。实际真正出社会才发现自己什么都干不了，没人给你建议或者帮助，我只能自己找现有资源。实习期的前三个工作都是经过学校、老师、同学的介绍才能找的到。当时的自己连找工作从哪里下手都不知道。最后一份工作是自己在外面逛的时候才找到，当时的自己也是迷茫的很

# 4、打杂期

​ 4.1、去广州钟落潭镇干了三个月的广告设计

​ 4.2 、回到深圳继续干网优，工作经历跟之前实习差不多。干了差不多三个月

​ 4.3、感觉工作工资都很低，又很我爸干了四个月的工地，后面又因为吃不消，又又又跑了

总结：这期间也是一样迷茫，当时的自己也是想学一门技术，能多挣点钱。找这些工作也是看周边同学在干什么，自己能干什么来找的，我是什么资源都没得，没人给过什么工作参考建议。父母也不知道我能干什么，他们所了解的也是一些底层的工作和资源。这也是没得发的事，毕竟你出生就是这样。

# 5、发育期

​ 5.1、来到人生转折点的公司，去一家培训机构干网络管理员，在里面学习了软件开发、网络设备的管理、服务器的一些知识。大概干了一年多点，经过和同事的沟通知道有运维这个岗位。

​ 5.2、去了一家游戏设计的公司干网络运维，大概干了半年

​ 5.3、去了一家p2p公司在里面也是干的网络和运维、但实际职位是运维工程师

​ 5.4、到了一家搞医疗器械销售、公司以前还搞过设备租赁。实际工作内容也比较杂。什么都要干，网络、桌面、服务器、打印机。但实际公司没有什么业务量，干了9个月就倒闭了，也成功领取了人生第一笔补偿金。

​ 5.5、中间还去了一家搞外包的it软件公司，主要是社保业务相关的。

总结：关于我为什么干网络管理员也是在学校期间学了一些知识，实习的时候也去搞过网络宽带的安装，懂得一些基础的知识。在深圳有些小公司就要这些有基础的人员。因为便宜和干的事杂，这个工作也好找些。为什么稀里糊涂就干上了运维，这可能也有一些运气成分在里面，当你卡在这个不上不下的时期你必须去寻找突破口，当时的我面临两个选择，要么去干软件工程师，要么干网络运维。可惜软件工程师我学习了半年感觉还是吃力的很，所以就选择了运维。后面换了很多家公司也是为了学习工作技能经验，也是为了挣高工资做准备。毕竟当时以为个人能力一直呆在一家公司不足以维持生活（有了家庭、小孩、房贷）。都是生活逼得我向前，这才有了后面的专业期。

# 5、专业期

5.1、在深圳西丽去了一家搞跨境电商的公司，在里面学习蛮多的知识，对以后的工作有很多帮助

5.2、在福田保税区去了一家搞银行业务相关的公司，在里面负责中台相关的，在里面学习了很多架构相关的知识

5.3、去了一家搞金融，在这个公司里面才是大展拳脚的地方。搞了很多自己想搞的事情，但是毕竟一个人能力有限，最后还是遇到瓶颈就跑了，这次是有些心累想回老家看看，离开了深圳

5.4、现在这家公司是干b端业务，主要是从一些大型的公司手中接到项目，简称包中包。我负责银行这边系统的运维。

总结：专业期的好处不言而喻，那就是工资可观，能够维持家庭的正常运转。但所付出的代价也是惨痛的，每天基本是早出晚归，责任肯定是比发育期要大很多，工作就是这样不可能事事如意，愿未来我能不为生计所迫，能干想干的事，不想干的事就远离。
