# 修改源方法

**临时使用**

```bash
#可在使用 pip 的时候在后面加上 -i 参数，指定 pip 源

pip install scrapy -i https://pypi.tuna.tsinghua.edu.cn/simple
```

**永久修改**

**Linux**

使用文本编辑工具打开 ~/.pip/pip.conf 文件 (如果没有当前文件夹或文件，请创建后操作) ，修改文件内容如下：

```bash
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
```

**Windows**

WIN+R 打开用户目录 %HOMEPATH%，在此目录下创建 pip 文件夹，在 pip 目录下创建 pip.ini 文件，内容如下:

```bash
[global]
index-url = https://pypi.tuna.tsinghua.edu.cn/simple
```

# 下载离线安装包

要在 Linux 服务器上离线安装特定版本的 pandas 模块，您可以按照以下步骤操作：

1. 在有网络连接的计算机上下载 pandas 2.1.4 及其依赖项：
   
   使用 pip 下载包及其依赖项：
   
   ```
   pip download pandas==2.1.4
   ```
   
   这将下载 pandas 2.1.4 及其所有依赖项到当前目录。

2. 将下载的文件传输到 Linux 服务器：
   
   使用 scp、sftp 或其他文件传输方法将下载的 .whl 文件传输到 Linux 服务器。

3. 在 Linux 服务器上安装 pandas：
   
   进入包含下载文件的目录，然后使用 pip 安装：
   
   ```
   pip install pandas-2.1.4-*.whl --no-index --find-links .
   ```
   
   这条命令会安装 pandas 2.1.4 及其依赖项。

4. 验证安装：
   
   在 Python 环境中导入 pandas 并检查版本：
   
   ```python
   import pandas as pd
   print(pd.__version__)
   ```
   
   应该显示 2.1.4。

如果您当前安装的是 2.2.3 版本，这个过程会将其降级到 2.1.4。

请注意，降级可能会影响依赖于较新 pandas 版本的其他包。在降级之前，建议检查您的项目是否与 pandas 2.1.4 兼容。
