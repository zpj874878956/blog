# 问题分析

根据提示是 /usr/lib/x86_64-linux-gnu/ 路径下的 libstdc++.so.6 缺少版本 GLIBCXX_3.4.29

sudo find / -name "libstdc++.so.6*"

**4、将此处的文件拷贝到我们需要的文件夹内，删除原来的链接，并建立起新的**

```bash
cp /root/miniconda3/lib/libstdc++.so.6.0.29 /usr/lib/x86_64-linux-gnu/
rm /usr/lib/x86_64-linux-gnu/libstdc++.so.6ln -s /usr/lib/x86_64-linux-gnu/libstdc++.so.6.0.29 /usr/lib/x86_64-linux-gnu/libstdc++.so.6


```
