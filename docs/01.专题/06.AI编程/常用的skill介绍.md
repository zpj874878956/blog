openclaw技能集

1、使用 summarize CLI 对 URL 或文件进行汇总（网页、PDF、图像、音频、YouTube）。

2、使用 Obsidian Vault（纯 Markdown 笔记）并通过 obsidian-cli 实现自动化。

3、tavily-search通过 Tavily API 实现 AI 优化的网络搜索。为 AI 代理返回简洁、相关的搜索结果。

4、baoyu-skills分享的 Claude Code 技能集，提升日常工作效率。

cladue code技能集

1、article-extractor-summary（文章提取和总结的skill）

2、lark（飞书的skill，跟飞书语音文档，飞书的群聊一些交互）

3、skill-writer（写作的skill）

4、subtitle-proofreader（标题生成的skill）

5、wechat-title-generator（微信文章创建的skill）

6、n8n-mcp、n8n-skills(https://github.com/czlonkowski/n8n-mcp)
