Claude Code 共有 **3 种工作模式**：

|模式|说明|
|---|---|
|**Normal（普通模式）**|默认模式，Claude 会请求权限后执行操作|
|**Plan（计划模式）**|只分析和规划，不执行任何实际操作，适合先review再执行|
|**Auto-accept（自动接受模式）**|自动批准所有操作，无需手动确认，适合信任的任务|

---

## Windows 切换命令

**方式一：键盘快捷键**

```
Shift + Tab     # 循环切换三种模式（Windows 上可能有 bug）
Alt + M         # 备用快捷键
```

**方式二：斜杠命令（最稳定）**

```
/plan           # 切换到 Plan 模式
/auto           # 切换到 Auto-accept 模式
/normal         # 切换到 Normal 模式
```

**方式三：启动时指定模式**

```bash
claude --plan           # 以 Plan 模式启动
claude --auto-accept    # 以 Auto-accept 模式启动
```

**方式四：配置文件设置默认模式**（`.claude/settings.json`）

```json
{
  "permissions": {
    "defaultMode": "plan"
  }
}
```

---

> **推荐**：Windows 上优先用斜杠命令 `/plan` `/auto` `/normal`，最稳定可靠。
