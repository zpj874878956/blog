# 引言

> 蓑衣圆子是湖北省黄梅县的特色美食。它口感爽滑、甜而不腻、营养丰富、老少皆宜。

# 制作方式

### 一.原料:

 芝麻、猪板油、白砂糖、芋头、红薯粉。

### 二.做法:

1.将芝麻炒熟并粉碎。

2.生板油去筋，取其精华部分并用手撕碎。

3.将粉碎好的芝麻和撕碎好的猪油加上白砂糖一起混合，捏成鸡蛋大的圆子待用。

 4.将芋头蒸熟去皮，加上红薯粉充分混合揉成面团。

5.用第四步做好的面团按成薄薄的皮，抱住第三步中的圆子，做成圆子，以此类推，将所有做好的圆子放入蒸锅，蒸熟即可食用。

### 三.象征意义:

 圆子意味着团团圆圆，而且有芋头，象征着“遇”圆，是遇到就圆满的意思。
