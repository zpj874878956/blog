# 引言

> 捶肉又叫杵肉，糊艿肉。每当有节日，在黄梅城乡许多家庭的厨房都会传来“砰、砰、砰”木捶敲打的声音，那就家庭主妇们正在用小木捶在捶肉。这捶肉啊不但味道鲜美，做起来也很简单，是许多主妇的拿手菜，也是黄梅传统菜谱上的一道特色菜。

# 制作方式

### 一.原料：

上好的纯瘦肉、苕粉、马蹄、葱花与姜末

### 二.做法:

1.肉要裹着纯正的苕粉一起锤，把苕粉锤到肉里面去

2.煮好的高汤，最好是土鸡汤来下肉片

3.马蹄切成沫或者片，先下锅煮熟再下肉片，但不能太多

4.千万不要煮过头，越嫩越好，再配上必须的佐料，葱花与姜末不可少
