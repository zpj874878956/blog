# kuberneters面试题

#### 创建pod流程

- 先编写yaml文件，apiserver做鉴权、认证、准入，将对象数据写入到etc

- deployment-controller观察期望的副本数，创建更新replicaset

- replicaset-controller创建pod对象

- scheduler为每个pod选取节点
- 目标节点通过kubelet创建容器、挂载卷等等
