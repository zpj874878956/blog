# 一、软件培训（贷款）
1、在招聘软件上打着高薪招工的幌子，把你叫过去
2、实际hr（其实就是销售员）把你拉到一个小黑屋疯狂输出洗脑，还有专门的培训师洗脑
3、然后给你开贷款，做软件培训，培训完自己去找工作。（不要信他们推荐工作，实际还是自己找）

## 类似的贷款经历
招工叫你贷款买车送货
买手机、电动车贷款（这种实际还是你自己没钱，想超前消费，别人给你提供门路，你多付点钱）

# 二、租房
1、各类租房软件上的信息（咸鱼、58等等）
2、房子图片跟价格不对称，诱导你看房
3、看到合适的还要收取你中介费（实际租房信息没有写）

# 三、新手找工作
1、在没提高你的工作技能和认知的情况下，请不要相信任何高薪工作。
2、任何交钱、压身份证、高薪骗你去国外打工的都是诈骗，轻者骗钱，重者有生命安全。
3、新手找工作不要眼高手低，不要一开始就想着高薪，天上不会掉馅饼，要放下身段去学习，后面积累一定经验，慢慢就能挣到你想要的。你的认知决定你能赚多少钱。
3、提高工作技能和认知需要时间，前方路很长，请有耐心慢慢熬，后面都会慢慢有的。
