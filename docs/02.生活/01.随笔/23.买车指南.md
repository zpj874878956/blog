# 先了解自己需要购买的车型

- 通过汽车之家、懂车帝
- 上网查找各种汽车相关知识
- 查找附近的4s店

# 了解自己所需购买的车型

- 新能源
- 油车
- 混动车
- 了解各个车型的类别，整理表格

# 实体店打听价格

- 采用迂回战术，战术性的了解，假装了解一个底价，打听对方底价
- 用对方的底价探听另一边的底价

# 羊毛出在羊身上

- 一般给你优惠的价格都是在各个功能挤出来，例如：贷款
- 贷款的套路深，一般先不要交定金，防止在签合同出幺蛾子
- 不要相信销售的话，销售的话听一半就行
- 一定不要交定金和诚意金
