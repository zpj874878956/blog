# 1、高中时期

​    高一下学期叛逆不想读书，跟父母去深圳打工，在工地搬砖，主要是搬运水泥和搅拌水泥浆，协助帮忙做苦力活。干过一个多月感觉吃不消放弃了就回到学校去继续读书

# 2、大学时期

2. 1、做过派发传单、在校门口摆摊卖苹果、国庆期间在酒店厨房做传菜员。

2. 2、暑假期间在深圳做过两个月保安。其它同学去工厂了我没有去。

# 3、实习期

​    3.1、在武汉做不到一个月的网优工程师，就是去一些基站和大型商超测试信号、去一些工厂把影响信号源的私人装的信号源设备偷偷装过滤波源的装置。

​    3.2、去深圳工厂干了三个月的流水线

​    3.3、去浙江绍兴公司干了半年，主要业务是网上卖菜，在里面做销售、一些其它杂事。

​    3.4、考了一个驾照

​    3.5、去中山干了半年的宽带安装

# 4、打杂期

​    4.1、去广州钟落潭镇干了三个月的广告设计

​    4.2 、回到深圳继续干网优，工作经历跟之前实习差不多。干了差不多三个月

​    4.3、感觉工作工资都很低，又很我爸干了四个月的工地，后面又因为吃不消，又又又跑了

# 5、发育期

​    5.1、来到人生转折点的公司，去一家培训机构干网络管理员，在里面学习了软件开发、网络设备的管理、服务器的一些知识。大概干了一年多点，经过和同事的沟通知道有运维这个岗位。

​    5.2、去了一家游戏设计的公司干网络运维，大概干了半年

​    5.3、去了一家p2p公司在里面也是干的网络和运维、但实际职位是运维工程师

​    5.4、到了一家搞医疗器械销售、公司以前还搞过设备租赁。实际工作内容也比较杂。什么都要干，网络、桌面、服务器、打印机。但实际没有什么业务量

​    5.5、中间还去了一家搞外包的it软件公司，主要是社保业务相关的。

# 5、专业期

   5.1、在深圳西丽去了一家搞跨境电商的公司，在里面学习蛮多的知识，对以后的工作有很多帮助

   5.2、在福田保税区去了一家搞银行业务相关的公司，在里面负责中台相关的，在里面学习了很多架构相关的知识

   5.3、去了一家搞金融，在这个公司里面才是大展拳脚的地方。搞了很多自己想搞的事情，但是毕竟一个人能力有限，最后还是遇到瓶颈就跑了，这次是有些心累想回老家看看，离开了深圳

   5.4、现在这家公司是干b端业务，主要是从一些大型的公司手中接到项目，简称包中包。我负责银行这边系统的运维。
