# 引言
> 认知决定了一个人对世界的理解方式，进而决定了他在生活和工作中看到什么、怎么反应、能走多远。

# 从学校到社会的过度期认知
1、这个时候你，实际还是懵懂期，就跟你刚出生那会，啥都不会，找工作都不知道从哪开始。
2、我刚出社会那会，都不知道有招聘软件（后面想想在学校时候听老师提过一句，那时候上课的你没当回事）
，找工作都是去人才市场、或者路边贴的字条招工、再或者熟人介绍。这类找的工作其实都不好，都是底层，
但是，可以做一段时间，这段经历其实在后面生活中可以提高你的认知的。

# 读书其实可以改变认知
1、你考试的分数决定你要上的学校、决定你认识的人，学校再决定你做的什么工作
2、上的大专就会被安排工厂流水线，上的本科校招的有可以去国企或者事业单位工作机会

# 家庭认知
1、家里人从小到大给你灌输的知识，也在后面工作生活决定着你的认知。

# 成年人要为自己的认知买单
因为认知决定了决策，决策带来了结果，而结果是自己必须承担的。别人可以帮你一时，但长期来看，没人能替你为自己的认知水平负责。
**认知越低，代价越高；认知越高，活得越轻松。**
