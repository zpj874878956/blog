# 1、[star-history](https://star-history.com/)
在线查看 GitHub 项目的 Star 增长趋势的网站，并生成趋势图，可以嵌入到你的开源项目

# 2、[gh-card.dev](https://gh-card.dev/)
免费为你的 Github 开源项目生成卡片的网站

# 3、[git教程](https://nulab.com/zh-cn/learn/software-development/git-tutorial/git-basics/)
猴子都能懂的GIT入门
# 4、[开源软件指南](https://opensource.guide/zh-hans/)
开源软件是由像你这样的人开发的。了解一下如何启动和发展您的项目

# 5、[gitignore生成器](https://www.toptal.com/developers/gitignore/)
为你的项目创建必要的 .gitignore 文件（忽略某些大文件不提交到仓库）

# 6、[github个人主页生成器](https://githubprofile.com/zh)
使用 GitHub 个人主页生成器创建精彩的 GitHub 个人主页。使用 GitHub 个人主页生成器轻松展示您的技能和项目。

# 7、[github-cheat-sheet](https://github.com/tiimgreen/github-cheat-sheet)
Git 和 GitHub 的酷炫功能列表。

# 8、[act](https://github.com/nektos/act)
假如你有 GitHub Actions 需要更高效的调试效果，可以试试 act 这个本地运行 GitHub Actions 的工具，安装后直接读取 .github/workflows/ 里面的文件去执行，相比每次改动去看云端的变化可以提高开发效率。

# 9、github action教程GitHub Actions 入门教程 - 阮一峰的网络日志
[文章链接](https://www.ruanyifeng.com/blog/2019/09/getting-started-with-github-actions.html)

# 10、[github开发教程](https://howtosos.eryajf.net/)
基础内容->github action->github tips
