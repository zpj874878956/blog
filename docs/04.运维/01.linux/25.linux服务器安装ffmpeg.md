# 一、下载官方的 ffmpeg

**下载地址：https://ffmpeg.org/**

![418c7c85234b831f.png](https://pdf.chatquanttail.top/2024/08/418c7c85234b831f.png)

在服务器执行以下命令

```bash
wget https://johnvansickle.com/ffmpeg/releases/ffmpeg-release-amd64-static.tar.xz
tar -xf ffmpeg-release-amd64-static.tar.xz
mv ffmpeg-7.0.1-amd64-static ffmpeg
```

**添加环境变量**

```bash
FFMPEG_HOME=/usr/local/ffmpeg
export PATH=$PATH:$FFMPEG
```

# 源码编译安装

在终端中执行以下命令安装lame和yasm：

```bash
yum install -y lame yasm
```

在官网下载对应的源码包

```bash
wget https://ffmpeg.org/releases/ffmpeg-7.0.2.tar.xz
tar -xf ffmpeg-7.0.2.tar.xz
mv ffmpeg-7.0.2.tar.xz ffmpeg
```

执行编译安装命令

```bash
cd ffmpeg
./configure
make && make install
```

# 使用ffmpeg

1. 查看视频信息

```bash
ffprobe input.mp4
```

2. 转换视频格式

```bash
ffmpeg -i input.mp4 output.avi
```

3. 截取视频片段

```bash
ffmpeg -i input.mp4 -ss 00:00:10 -t 00:00:05 output.mp4
```
