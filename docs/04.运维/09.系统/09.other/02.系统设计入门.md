## 1.**[The System Design Primer](https://github.com/donnemartin/system-design-primer)**

- **内容**：系统设计基础，涵盖高可用性、可扩展性、负载均衡、数据库分片等核心概念。
- **适用人群**：想学习系统设计基础的开发者、面试准备者。
- **特点**：结构清晰，适合从零开始学习，包含大量实践案例。

## 2.**[System Design 面试指南](https://github.com/Admol/SystemDesign)**

- **内容**：系统设计面试常见问题解析，包括设计高并发系统、分布式存储、消息队列等。
- **适用人群**：正在准备大厂系统设计面试的开发者。
- **特点**：针对面试场景，提供高频问题及解答思路，适合短时间内提升面试能力。

## 3.**[system-design-101](https://github.com/ByteByteGoHq/system-design-101)**

- **内容**：以图文结合的方式讲解系统设计基础概念，如负载均衡、数据库分片、缓存策略等。
- **适用人群**：初学者、希望通过可视化方式理解系统设计的开发者。
- **特点**：通俗易懂，适合入门，降低理解难度。

## 4.**[OceanBase ODC](https://github.com/oceanbase/odc)**

- **内容**：企业级数据库协同开发平台，支持 SQL 规范检查、数据管理、权限控制等。
- **适用人群**：数据库开发者、企业级应用开发者。
- **特点**：支持 OceanBase、Oracle、MySQL 等多种数据库，提升 SQL 开发效率，适用于企业数据管理。
