# 使用innobackupex备份

#### 全量备份操作步骤

```bash
innobackupex /data/mysql-backup/blog --user=root --password=Tencent@123 --socket=/data/database/mysql/mysql.sock
```

#### 恢复操作

**停止mysqld服务**

```shell
systemctl stop mysqld
```

**清空数据目录**

```shell
rm -rf /data/database/mysql/*
```

**恢复备份**

```shell
innobackupex  --defaults-file=/etc/my.cnf --copy-back /data/mysql-backup/blog/2024-10-18_16-03-27 --user=root --password=Tencent@123 --socket=/data/database/mysql/mysql.sock
```

**在恢复数据后，应用备份日志，以确保数据库处于一致状态：**

```bash
innobackupex --apply-log /data/mysql-backup/blog/2024-10-18_16-03-27 --user=root --password=Tencent@123 --socket=/data/database/mysql/mysql.sock
```

**更改数据目录权限**

```bash
chown -R mysql:mysql mysql
```

**启动mysql服务**

```bash
systemctl start mysqld
```

#### 增量备份操作步骤

在执行增量备份之前，首先需要进行一次全量备份，并将其作为基础。以下是进行全量备份的命令：

```bash
innobackupex /data/mysql-backup/blog --user=root --password=Tencent@123 --socket=/data/database/mysql/mysql.sock
```

**进行增量备份**

```bash
innobackupex --user=root --password=Tencent@123 --socket=/data/database/mysql/mysql.sock --incremental --incremental-basedir=/data/mysql-backup/blog/2024-10-18_16-03-27 /data/mysql-backup/zl
```

**应用增量备份日志**

在完成增量备份后，使用 `innobackupex --apply-log` 命令应用增量备份的日志，以确保数据一致性。

```bash
innobackupex --apply-log /data/mysql-backup/zl --user=root --password=Tencent@123 --socket=/data/database/mysql/mysql.sock
```

**恢复增量备份**

**停止mysqld服务**

```shell
systemctl stop mysqld
```

**清空数据目录**

```shell
rm -rf /data/database/mysql/*
```

**恢复全量备份**

```shell
innobackupex  --defaults-file=/etc/my.cnf --copy-back /data/mysql-backup/blog/2024-10-18_16-03-27 --user=root --password=Tencent@123 --socket=/data/database/mysql/mysql.sock
```

**在恢复数据后，应用备份日志，以确保数据库处于一致状态：**

```bash
innobackupex --apply-log /data/mysql-backup/blog/2024-10-18_16-03-27 --user=root --password=Tencent@123 --socket=/data/database/mysql/mysql.sock
```

**恢复增量备份**

```shell
innobackupex  --defaults-file=/etc/my.cnf --copy-back /data/mysql-backup/zl/2024-10-18_18-11-07 --user=root --password=Tencent@123 --socket=/data/database/mysql/mysql.sock
```

**应用增量备份日志**

```bash
innobackupex --user=root --password=Tencent@123 --socket=/data/database/mysql/mysql.sock --apply-log /data/mysql-backup/blog/2024-10-18_16-03-27 --incremental-dir=/data/mysql-backup/zl/2024-10-18_18-11-07
```

**更改数据目录权限**

```bash
chown -R mysql:mysql mysql
```

**启动mysql服务**

```bash
systemctl start mysqld
```
